# Project-Race
MAR Project : race game using the ThreeJS library. 

## Usage

Open the ```marace.10.3.html``` file in your browser.

## Controls
- ```space``` key : jump
- ```backspace``` key : reset
- ```p``` key : change the camera mode
- ```n``` key : change the vehicle & reset
- ```z```,```q```,```d```,```s``` keys : moving/braking the vehicle
